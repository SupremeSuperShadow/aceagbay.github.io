Pixel Dungeon

A procedural 2D dungeon crawler.

Controls

  • WASD / Arrow Keys — Movement
  • Space — Dodge
  • Mouse — Aim
  • Left Click — Shoot
  • Esc / P — Pause

Overview

This project was made to display knowledge of FSM, tilesets, 2D graphics, signals, gameplay loop, and sound design.

Tools

  • Godot v4.5.1
  • Pixelorama
  • Audacity
  • BeepBox
