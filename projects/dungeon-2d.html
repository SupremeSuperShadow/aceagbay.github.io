Pixel Dungeon

A procedural 2D dungeon crawler.

Controls

  • WASD / Arrow Keys — Movement
  • Space — Dodge
  • Mouse — Aim
  • Left Click — Shoot
  • Esc / P — Pause

Tools

  • Godot v4.5.1
  • Pixelorama
  • Audacity
  • BeepBox

Contribution

  • Sole developer

      • Highlights

      • Procedural room generation with dynamic obstacles and enemies
      • Signal-driven game flow (doors, damage, win/lose states)
      • Finite State Machine determines enemy behavior
      • Shader-based enemy vulnerability and player/enemy visual feedback
      • Settings menu with live sliders and shortcut buttons
      • Level entry animation to improve player orientation under pressure
