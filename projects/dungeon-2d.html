Pixel Dungeon

A procedural 2D dungeon crawler focused on combat flow and replayability.

Controls

  • WASD / Arrow Keys — Movement
  • Space — Dodge
  • Mouse — Aim
  • Left Click — Shoot
  • Esc / P — Pause

Tools

  • Godot v4.5.1
  • Pixelorama
  • Audacity
  • BeepBox

Contribution

  • Sole developer

Highlights

  • Procedurally generated rooms with changing enemies and obstacles
  • Enemy AI driven by a finite state machine
  • Doors, damage, and win/lose conditions coordinated through a signal system
  • Visual feedback for damage and enemy vulnerability using custom shaders
  • Level entry animation that helps orient the player before combat begins
