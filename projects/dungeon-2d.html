2D Dungeon Crawler

A procedural 2D action RPG with replayable dungeon layouts.

Controls

  • WASD / Arrow Keys — Movement
  • Mouse / Keyboard — Actions

Overview

This project began as an experiment in procedural generation and grew into a complete, playable game.

Problems Solved

  • Procedural room connectivity
  • Preventing soft-locks in generated layouts
  • Managing combat state transitions

Systems & Architecture

The game is built around modular systems for dungeon generation, enemy behavior, and animation state management.

Challenges & Trade-offs

A major challenge was balancing flexibility with simplicity in the dungeon generator.

What I’d Improve Next

  • Refactor input handling into a unified system
  • Improve performance for larger dungeon sizes
